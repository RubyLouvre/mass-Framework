$.fn.map(callback)

描述：

对实例中的每一个元素进行map操作，将得到结果进行过滤，只留下是节点或window对象的元素，构成新的mass实例返回。

参数：

callback
必需。回调函数。

返回值：

原mass实例。

例子

更换innerHTML

  • 列表1
  • 列表2
  • 列表3
  • 列表4
      <ul class="container">
        <li class="inner" >列表1</li>
        <li class="inner" >列表2</li>
        <li class="inner" >列表3</li>
        <li class="inner" >列表4</li>
      </ul>
            
                 $.require("ready,node",function(){
                   var neo =  $(".container li").map(function(el,i){
                        return el.firstChild;
                    }) ;
