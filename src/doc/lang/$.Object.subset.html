object.subset(arr)

描述：

根据传入数组取当前对象相关的键值对组成一个新对象返回。

arr
必需。新对象的键名。
例子 
                $.require("lang",function(){
                    var obj = {
                        a:"one",
                        b:"two",
                        c:"three"
                    };
                    var neo = $.lang(obj).subset(["a","b"]);
                    $.log($.dump(neo), true);
                });
