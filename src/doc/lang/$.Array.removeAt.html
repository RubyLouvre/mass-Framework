array.removeAt(index)

描述：

破坏性方法，修改原对象。移除指定位置上的元素。

index
必须。Number。要移除元素的索引值。

返回值：

被移除的元素或null。

例子 
                $.require("lang",function(){
                    var test = [3, 5, 6, 1, 20];
                    var a = $.lang(test);
                    $.log(a.removeAt(2),true);
                    $.log($.dump(test),true);
                });
