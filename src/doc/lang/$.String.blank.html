string.blank()

描述：

判断字符串是否全部由空白字符串组成或者一个字符也没有。

例子 
                $.require("lang",function(){
                    var a = $.lang("  ");
                    $.log(a.blank(),true);
                    var b = $.lang("\n");
                    $.log(b.blank(),true);
                    var c = $.lang(" ");
                    $.log(c.blank(),true);
                    var d = $.lang("\xa0");
                    $.log(d.blank(),true);
                    var e = $.lang("");
                    $.log(e.blank(),true);
                });
