$.slice(arr,[start],[end])

描述：

对类数组对象进行数组化，如果传入超个一个以上的参数，则其行为相当于数组的slice方法一样，进行切片。

所谓类数组对象通常指arguments对象，字符串对象，NodeList节点集合，由于这里没有强化对类型的检测，因此只要拥有length属性就被认为是类数组 因此牵及的对象增添window,function,form元素,select元素等东西.

参数：

arr
必需。ArrayLike。要处理的类数组对象。
start
可选。要抽取的片断的起始下标。如果是负数，从后面取起。
end
可选。规定从何处结束选取。

返回值：

Array

例子 
                $.require("lang",function(){
                    var ret = $.slice({0:'aa',1:'bb',2:'cc',length:3});
                    ret = $.dump(ret);
                    $.log(ret,true);
                });
