$.unbind(obj,type,fn,[ phase ])

描述：

与$.bind配套使用，用于卸载原生事件发送器上的某个事件回调，注意它没有返回值，因为没有意义。

参数：

obj
必需。原生的事件发送器，如window,document与元素节点。
type
必需。String。事件类型，注意不要使用onXXX的格式。
fn
必需。Function。
phase
可选。Boolean。是否使用捕获，默认false，与不支持捕获的IE678保持一致。
例子 
            var i = 0;
            var fn =  $.bind(document,"click",function(){
                i ++;
                $.log("触发了第"+i+"点击事件",true);
                if(i === 4){
                     $.unbind(document,"click",fn);
                      $.log("已移除点击事件",true);
                }
            })
