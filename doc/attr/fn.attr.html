$.fn.attr( name, [val] )

描述：

对元素节点的自定义属性进行读写操作。

参数：

  • 如果只有一个字符串参数，则对第一个匹配节点进行读取操作。
  • 如果有两个参数，第一个为字符串，第二个参数不为null, 则对所有匹配节点进行赋值操作，如果第二个参数为null则进行移除属性的操作。
  • 如果只有一个对象参数，则依取取得里面的键值对，对所有匹配节点进行多个赋值操作。
例子

为页面中所有img元素设置一些属性。

set by attr API 


        

            
            
            
            
set by attr API

        

    

$.require("ready,attr", function() {
    $(".sample1 img").attr({
        src: "/styles/attr_rabbit.jpg",
        title: "流氓兔",
        alt: "流氓兔"
        width: "200px",
        height: "200px"
    });
    $(".sample1 strong").text($("img").attr("alt"));
});

使第二个后面的按钮disabled

    $(".sample2 input").gt(1).attr("disabled", "disabled");
