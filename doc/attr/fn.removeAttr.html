$.fn.removeAttr( attrName )

描述：

为匹配的每一个元素节点移除指定的属性。

.removeAttr() 方法使用原生的 JavaScript removeAttribute() 函数,但是它的优点是能够直接被jQuery对象访问调用。而且具有良好的浏览器兼容性。

参数：

attrName
必需。String。要移除的属性名。
例子

让下面的文本域可以编辑 。


    

        
    

    
                
$.require("ready,attr,event", function() {
    $(".sample1 input").removeAttr("disabled").focus().val("editable now");
});
