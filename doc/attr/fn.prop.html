$.fn.prop( name, [val] )

描述：

对元素节点的固有属性进行读写操作。

元素节点的固有属性是无法删除的，只能还原为初始值，如className属性的初始值为空字符串， 布尔属性为false, form元素的method属性为get, input元素的type属性为text, script元素的type属性为text/script, style元素的media属性为all.... 因此对于表单元素的checked, selected, disabled, readonly等属性操作，我们应该使用prop方法而不是attr方法。

accept, accept - charset, align, alink, axis,
bgcolor, charset, checked, class, clear, codetype, color,
compact, declare, defer, dir, direction, disabled,
enctype, face, frame, hreflang, http - equiv, lang,
language, link, media, method, multiple, nohref,
noresize, noshade, nowrap, readonly, rel, rev,
rules, scope, scrolling, selected, shape, target,
text, type, valign, valuetype, vlink

参数：

  • 如果只有一个字符串参数，则对第一个匹配节点进行读取操作。
  • 如果有两个参数，第一个为字符串，第二个参数不为null, 则对所有匹配节点进行赋值操作，如果第二个参数为null则进行移除属性的操作。
  • 如果只有一个对象参数，则依取取得里面的键值对，对所有匹配节点进行多个赋值操作。
例子

为页面中所有img元素设置一些属性。


    

        
        
        

    

    

$.require("ready,attr,event", function() {
    $(".sample1 input").change(function() {
        var $input = $(this);
        $(".sample1 div").html(".attr('checked'): <b>" + $input.attr('checked') + "</b><br>" + ".prop('checked'): <b>" + $input.prop('checked') + "</b><br>")
    }).change();
});

$.require("ready,attr", function() { //禁用复选框
    $(".sample2 input[type='checkbox']").prop({
        disabled: true
    });
});

$.require("ready,attr", function() { //启用复选框
    $(".sample2 input[type='checkbox']").prop({
        disabled: false
    });
});
