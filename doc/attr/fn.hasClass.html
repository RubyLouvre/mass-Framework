$.fn.hasClass(className)

描述：

检查匹配的元素是否含有某个特定的类名。

参数：

className
必需。String。要查询的类名。

返回值：

Boolean

例子
请点击它

                <div class="sample">请点击它</div>
                <style>
                    .selected {
                       background:#8B0000;
                    }
                </style>
    

$.require("ready,event,attr", function() {
    $(".sample").click(function(e) {
        var self = $(this);
        if (self.hasClass("selected")) {
            self.removeClass("selected");
        } else {
            self.addClass("selected");
        }
    });
});
