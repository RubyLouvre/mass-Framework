$.parseData(elem, name)

描述：

取得元素的某一属性，转换为JSON数据。此API是框架用于转换HTML5的data-*属性的。

转换规则：

  • "true", "false", "null" 转换true,false,null
  • 形如数字的字符串转为数字
  • 如果是用花括号包起来的尝试转换为对象
  • 其他原样输出

参数：

elem
必需。元素节点。
name
必需。String。属性名。注意，无需带data-前缀，它会自行添加的。

返回值：

经转换后的数据。

例子 

        

    

$.require("ready,data", function() {
    var el = $("#test")[0]
    var data = $.parseData(el, "obj");
    $.log($.type(data), true);
    $.log($.dump(data), true);
    var array = $.parseData(el, "arr");
    $.log($.type(array), true);
    $.log($.dump(array), true);
    var nan = $.parseData(el, "nan");
    $.log($.type(nan), true);
    $.log($.dump(nan), true);
    var str = $.parseData(el, "str");
    $.log($.type(str), true);
    $.log($.dump(str), true);
});
