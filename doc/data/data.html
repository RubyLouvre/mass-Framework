$.data(obj, name , value)

描述：

为obj对象储存一个名为name，值为value的数据。

参数：

obj
必需。不为属性节点或文本节点的其他类型节点，文档对象，window对象，自定义对象等等。
name
必需。String。数据的名字。
value
必需。不为undefined的任意类型数据。数据的值。

返回值：

value

例子 
$.require("data", function() {
    var val = $.data(document.body, "test1", [1, 2, 3]);
    $.log(document.body.uniqueNumber, true);
    $.log($.dump(val), true);
});

$.data(obj,object)

描述：

为obj添加同时添加多个数据。

参数：

obj
必需。见上。
object
必需。Object。一个数据包。

返回值：

obj在$.memcache或自身（如果是自定义对象）上的缓存体，它包含所有相关的数据。

例子 
$.require("data", function() {
    var val = $.data(document.body, {
        aa: "aa",
        bb: "bb"
    });
    $.log($.dump(val), true);
});

$.data(obj,name)

描述：

取得与obj相关的名为name的数据。

参数：

obj
必需。见上。
name
必需。String。目标数据名。

返回值：

name的值，没有返回undefined。

例子 
$.require("data", function() {
    var val = $.data(document.body, "aa");
    $.log(val, true);
});
