节点操作模块

描述：

mass Framework的 最重要的模块，提供节点的基本操作（查找，创建，插入，删除，复制）。它直接依赖于 四个模块:class(提供扩展机制),support（提供特征嗅深，以使用更好的原生API）,query（提供选择器）,data(复制事件时用到)。

API 解说
init 等同于$，集seletor，domReady， htmlParser于一身！
toString 将它的所有节点的tagName放到一个数组返回
valueOf 将它的所有节点放到一个数组返回
length 将它的所有节点的个数返回
mass 当前框架的版本号，等同于$.mass
slice 取得它的同一部分节点，封装成mass对象返回
get 将它的一部分节点或所有节点放进一个数组返回
eq slice方法的变种，将索引值等于传参的那个节点，封装成mass对象返回
gt slice方法的变种，将索引值大于传参的那些节点，封装成mass对象返回
lt slice方法的变种，将索引值小于传参的那些节点，封装成mass对象返回
first slice方法的变种，将它的第一个节点，封装成mass对象返回
last slice方法的变种，将它的最后一个节点，封装成mass对象返回
odd 将它的所有索引值为奇数的那些节点，封装成mass对象返回
even 将它的所有索引值为偶数的那些节点，封装成mass对象返回
each 对它的每一个节点放入回调中执行
map 对它的每一个节点放入回调中执行，并收集返回结果，再封装成mass对象返回
collect 对它的每一个节点放入回调中执行，并收集返回结果
clone 克隆一个mass对象，它里面节点都会被深克隆
html 取得第一个节点的innerHTML或设置所有节点的innerHTML
text 取得第一个节点的innerText或设置所有节点的innerText
push 模拟原生数组的push行为，但返回的是一个mass对象
unshift 模拟原生数组的unshift行为，但返回的是一个mass对象
pop 模拟原生数组的pop行为，但返回的是一个mass对象
shift 模拟原生数组的shift行为，但返回的是一个mass对象
splice 模拟原生数组的splice行为，但返回的是一个mass对象
sort 模拟原生数组的sort行为，但返回的是一个mass对象，sort默认是根据节点在DOM树上的上下左右而排序。
reverse 模拟原生数组的reverse行为，但返回的是一个mass对象
remove 将它包含的所有节点移出DOM树，并清理它们所引用的数据
empty 将它包含的所有节点的孩子移出DOM树，并清理那些孩子所引用的数据
append 在它所包含的所有节点的内部的后面插入新节点
prepend 在它所包含的所有节点的内部的前面插入新节点
before 在它所包含的所有节点的外部的前方插入新节点
before 在它所包含的所有节点的外部的后方插入新节点
replace 将它所包含的所有节点都替换成新节点
appendTo append的反转操作，原来是A插入到B，现在是B插入到A
prependTo prepend的反转操作，原来是A插入到B，现在是B插入到A
beforeTo before的反转操作，原来是A插入到B，现在是B插入到A
afterTo after的反转操作，原来是A插入到B，现在是B插入到A
replaceTo replace的反转操作，原来是A被B替换掉，现在是B被A替换掉
find 在当前找到的节点的孩子中，寻找匹配给定的CSS表达式的节点，封装成mass对象返回
filter 在当前找到的节点中，寻找匹配给定的CSS表达式的节点，封装成mass对象返回
not 在当前找到的节点中，寻找不匹配给定的CSS表达式的节点，封装成mass对象返回。not与filter是一组，而不是与is!
is 判定当前节点是否匹配给定的CSS表达式
has 在当前的节点中，往下遍历他们的后代，收集匹配给定的CSS表达式的节点，封装成新mass实例返回。
closest 在当前的节点中，往上遍历他们的祖先，收集最先匹配给定的CSS表达式的节点，封装成新mass实例返回。
index 返回指定节点在其所有兄弟中的位置
parent 在当前的节点中，收集其父节点，封装成新mass实例返回。
parents 在当前的节点中，往上遍历他们的祖先，收集匹配给定的CSS表达式的节点，封装成新mass实例返回。
parentsUntil 在当前的节点中，往上遍历他们的祖先直到某个位置，收集匹配给定的CSS表达式的节点，封装成新mass实例返回。
next 在当前的节点中，收集其最近的右边的元素节点，封装成新mass实例返回。
nextAll 在当前的节点中，收集其右边的元素节点，封装成新mass实例返回。
nextUntil 在当前的节点中，收集其右边的元素节点直接某个位置，封装成新mass实例返回。
prev 在当前的节点中，收集其左边的元素节点，封装成新mass实例返回。
prevAll 在当前的节点中，收集其左边的元素节点，封装成新mass实例返回。
prevUntil 在当前的节点中，收集其左边的元素节点直接某个位置，封装成新mass实例返回。
children 在当前的节点中，收集其子元素节点，封装成新mass实例返回。
siblings 在当前的节点中，收集其兄弟元素节点，封装成新mass实例返回。
contents 在当前的节点中，收集其后代中元素节点，封装成新mass实例返回。
data 为匹配的节点们绑定或读取数据。
removeData 为匹配的节点们移除数据。
