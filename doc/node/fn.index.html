$.fn.index( [el] )

描述：

从匹配的元素中搜索给定元素的索引值，从0开始计数。。

参数：

el
可选。CSS表达式，DOM元素，或者dom对象。

返回值：

  • 如果没有参数，返回第一元素位于其兄弟的位置
  • 如果参数是CSS表达式，则先构建一个mass实例，然后返回原实例中第一个元素在新实例中的位置
  • 如果参数是节点或mass实例，则返回节点（如果是mass实例则取其第一个元素）位于原实例的位置
  • 如果没有找到，返回-1

$.require("ready,css,event", function() {
    var self = $(".sample1 div").click(function() {
        var index = self.index(this);
        $(".show_index").text("点击第" + index + "个DIV");
    });
});
