$.fn.data(key,[ val ])

描述：

对实例中的每一个元素存入或读出指定数据。

参数：

key
必需。第一种情况，String。数据的名字。如果只有它一个参数，则进行读操作，返回第一个元素名为key的数据
必需。第一种情况，一个普通的Object，进行写操作，同时对所有匹配元素写入多个数据，返回实例本身。
val
可选。不为undefined的任意类型数据。数据的值，如果存在，则进行写操作，为其所匹配的所有元素存入这个数据，返回实例本身。
