$.fn.filter( expr )

描述：

取得当前匹配节点的所有匹配expr的节点，组成新mass实例返回。

参数：

expr
必需。String|Function。CSS表达式，或者是一个回调函数。

返回值：

新的mass实例

例子

取得.sample1所有索引值为偶数的LI元素，这经常用于商场或新闻中隔行变色效果。

$.require("ready,css", function() {
    $(".sample2 li").filter(function(el, i) {
        return $("strong", el).length == 1
    }).css("color", "red")
});

使用过滤函数，this是当时正在操作的DOM元素。参数有两个,el与this一样，i为el在mass实例的索引值。
