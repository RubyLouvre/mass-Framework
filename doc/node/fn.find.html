$.fn.find( expr )

描述：

取得当前匹配节点的所有匹配expr的后代，组成新mass实例返回。

参数：

expr
必需。String。CSS表达式。

返回值：

新的mass实例

例子

搜索.sample1所有后代的span元素，和效果与$(".sample1 span")一样

Hello, how are you?

Me? I'm good.


    

        
Hello, how are you?

        
Me? I'm good.

    

    

$.require("ready,css", function() {
    $(".sample1").find("span").css("color", "red")
});
