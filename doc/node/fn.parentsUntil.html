$.fn.parentsUntil( [expr] )

描述：

查找当前元素的所有祖先，直到遇到选择器匹配的元素为止，不包括那个匹配到的元素。

参数：

expr
可选，String，CSS表达式，用于确定到哪个祖先时停止匹配。

返回值：

新mass实例

例子

查找 <li class="item-a"> 的前辈，直到 <ul class="level-1"> 为止，并给它们设置红色背景。
