$.fn.contents( )

描述：

取得匹配元素的所有节点，去重排序组成新mass实例返回。

此方法与children方法相似，但它比后者能获取更多的节点，如文本节点，注释节点。如果是一个iframe元素则取得其对应的文档对象。

返回值：

新mass实例。
