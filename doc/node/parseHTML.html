$.parseHTML( html, doc)

描述：

一种高级的创建节点的方式。利用createContextualFragment或innerHTML将字符串转换成相应的节点，并且能同时设置样式与属性，在绝大多数情况下 效率比createElement高。有关innerHTML与createElement创建节点的效率可见 这里注意，它能执行元素的内联事件，如$.parseHTML("<img src=1 onerror=alert(22) />")。

参数：

html
必需。String。要转换为节点的字符串。
doc
可选。Document。

返回值：

文档碎片

例子 
$.require("ready,lang", function() {
    var str = "<h1>My First<strong>Heading</strong></h1><p>My first paragraph.</p><script>alert('司徒正美')</script>";
    document.body.appendChild($.parseHTML(str)) //alert "司徒正美"
});
