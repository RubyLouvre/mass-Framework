$.fn.menu

描述：

菜单插件。

参数：

data
必需。一个对象数组,里面的对象要求拥有html或sub属性,sub也是一个类似结构的对象数组。 想实现无限级联，就不断往里面套sub对象数组吧。
hover_class
可选。菜单项移上去时发生作用的类名,默认是hover。
direction
可选。可用值为"vertical"或"horizontal", 默认是"vertical"。

返回值：

mass实例

例子 

      <div class="black"></div>
            
            .black .menu_item{
                width:120px;
                padding-left:10px;
                height:24px;
                line-height:24px;
                border-bottom:1px solid #fff;
                background:#000;
                color:#fff;
            }
            .black .hover {
                background:#636!important;
                color:black;
            }
            
             $.require("ready,more/menu",function( ){
                $(".black").menu({
                    data: [
                        {
                            html:"随笔",
                            sub:[
                                {html:"生活"},
                                {html:"杂文"},
                                {html:"思考"}
                            ]
                        },
                        {
                            html:"编程工具",
                            sub:[
                                {html:"Vim"},
                                {html:"NetBeans"},
                                {html:"Git"}
                            ]
                        },
                        {
                            html:"编程语言",
                            sub:[
                                {html:"Web开发",sub:[
                                        {html:"HTML"},
                                        {html:"javascript"},
                                        {html:"CSS"}
                                    ]},
                                {html:"Python"},
                                {html:"Java语言"},
                                {html:"C/C++"},
                                {html:"Android开发"}
                            ]
                        },
                        {html:"操作系统",sub:[
                                {html:"IOS"},
                                {html:"window"},
                                {html:"ubuntu"}
                            ]}
                    ],
                    direction: "horizontal"
                });
            });
