$.fn.off(types,[selector],[ fn ])

描述：

mass Framework实现事件卸载的入口函数, 通过它实现unbind, die, undelegate。

参数：

types
可选。String.事件类型,想同时卸载多个事件，可心用空格或逗号隔开，如"error load"或"DOMMouseScroll,mousewheel"。
可选。Object.上面同时卸载多个事件，它们的回调都是相同的，如果相独自指定回调， 则使用此法，如{ click:clickFn, mouseover:mouseFn}。
selector
可选，String，用于移除事件代理的回调。
fn
可选，Function，要移除的回调函数。
