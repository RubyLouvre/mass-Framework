$.fn.mouseover([ fn ])

描述：

为 mouseover 事件绑定一个处理函数，或者触发元素上的 mouseover 事件。

这个函数的第一种用法是 .bind('mouseover', handler) 的快捷方式，第二种用法是 .fire('mouseover') 的快捷方式。

当鼠标指针进入元素内时，mouseover事件就会被发送到这个元素，任何HTML元素都可以接受此事件。

参数：

fn
可选。Funcion。回调函数。
例子

测试dispatch mouseover事件时,不会触发mouseenter事件(主要原因是webkit下 mouseenter事件是使用mouseover事件来模拟的)

$.require("ready,event", function() {
    $("#red").bind("mouseenter", function(e) {
        $.log("mouseenter回调", true)
    });
    $("#red").bind("mouseover", function(e) {
        $.log("mouseover回调", true)
    });

    $("#red").fire("mouseover")
})

正确的情况是只显示"mouseover回调"
