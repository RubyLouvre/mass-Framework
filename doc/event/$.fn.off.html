$.fn.off(types,[ fn ])

描述：

dom Framework实现事件卸载的入口函数, 通过它实现unbind, die, undelegate。

它支持多种形式的传参方式:

$.fn.on(obj)
$.fn.on(str, fn)
$.fn.on(str)

参数：

types
必需。String.事件类型,想同时卸载多个事件，可心用空格或逗号隔开，如"error load"或"DOMMouseScroll,mousewheel"。
必需。Object.上面同时卸载多个事件，它们的回调都是相同的，如果相独自指定回调， 则使用此法，如{ click:clickFn, mouseover:mouseFn}。
fn
可选，Function，要移除的回调函数。
