$.fn.mousemove( [ fn ] )

描述：

为 mousemove 事件绑定一个处理函数，或者触发元素上的 mousemove 事件。

这个函数的第一种用法是 .bind('mousemove', handler) 的快捷方式，第二种用法是 .fire('mousemove') 的快捷方式。

当鼠标指针在元素内移动时，mousemove事件就会被发送到这个元素，任何HTML元素都可以接受此事件。

当鼠标在某个元素上移动时，我们通常需要知道鼠标指针的坐标，事件对象提供了两组坐标属性，pageX与pageY是相对整个页面的坐标系，clientX与clientY是相对屏幕可视区的坐标系。

值得注意的是，mousemove事件是当鼠标指针移动时触发的，即使是一个像素。这意味着数百个事件在短时间内产生。 因此我们必须简化mousemove回调里面的DOM操作，比如加一个变量，当其余9时才执行DOM操作。另一个方案是，使用事件代理，减少mousemove监听器的数量。

参数：

fn
可选。Funcion。回调函数。
例子 
$.require("ready,event", function() {
    var first = $(".first")
    var last = $(".last")
    $(".move").mousemove(function(e) {
        var pageCoords = "( " + e.pageX + ", " + e.pageY + " )";
        var clientCoords = "( " + e.clientX + ", " + e.clientY + " )";
        first.text("( e.pageX, e.pageY ) - " + pageCoords);
        last.text("( e.clientX, e.clientY ) - " + clientCoords);
    });
});
