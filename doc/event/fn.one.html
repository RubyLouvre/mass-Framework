$.fn.one(types,[ fn ])

描述：

只允许被绑定的事件回调执行一次就自行卸载，如果想限制其他次数，请使用 on方法

参数：

types
如果是字符串则为事件类型，如果为对象则允许同时绑定多种事件，键为事件类型，值为回调函数。
fn
在types为对象时是可选，为字符串是必需，这时它是回调函数
例子

点击第一个按钮只会为自己加1，点击第二个按钮会为两者加1

请点击上面的方块。。。


            <div class="sample1">
                <button>Button #1</button>
                <button>Button #2</button>
                <div><span>0</span> button #1 clicks.</div>
                <div><span>0</span> button #2 clicks.</div>
            </div>
    

$.require("ready,event,css", function() {
    var n = 0;
    var divs = $(".sample1 div").one("click", function() {
        var index = divs.index(this);
        $(this).css({
            borderStyle: "inset",
            cursor: "auto"
        });
        $(".sample1").text("Div at index #" + index + " clicked." + "  That's " + ++n + " total clicks.");
    });
});
