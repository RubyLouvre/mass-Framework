$.fn.die( types, fn )

描述：

移除通过die绑定的事件回调。

参数：

types
可选。String。一个包含一个或多个JavaScript事件类型的字符串，比如"click"或"keydown,"或自定义事件的名称。没有则清空此元素所绑定的所有回调。
fn
可选。Funcion。回调函数。
例子
Click!
$.require("ready,event,fx", function() {
    function callback() {
        $(".hide_div").show(1000);
    }
    $("#bind").click(function() {
        $("#theone").live("click", callback).html("现在点击它下方会出现文字");
    });
    $("#unbind").click(function() {
        $("#theone").die("click", callback).html("现在点击它一点反应也没有");
        $(".hide_div").hide(1000);
    });
});
