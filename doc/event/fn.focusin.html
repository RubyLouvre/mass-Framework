$.fn.focusin( fn )

描述：

为 focusin 事件绑定一个处理函数。

这个函数是 .bind('focusin', handler) 的快捷方式。

focusin 事件会在元素（或者其内部的任何元素）获得焦点时触发。这跟 focus 事件的显著区别在于， 它可以在父元素上检测子元素获得焦点的情况(换而言之，它支持事件冒泡)。

参数：

fn
必需。Funcion。回调函数。
例子 
$.require("ready,event,fx", function() {
    $(".sample p").focusin(function(e) {
        $(this).find("span").html("fire " + e.type).show().fadeOut(1000);
    });
});

$.require("ready,event", function() {
    $("#focusin").bind("focusin", function(e) {
        $("#focusin_text").text(e.type)
        e.target.style.color = "red";
    });
    $("#focusin").bind("focusout", function(e) {
        $("#focusin_text").text(e.type)
        e.target.style.color = "";
    });
})
