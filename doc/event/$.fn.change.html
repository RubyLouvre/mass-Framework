$.fn.change([ fn ])

描述：

为 change 事件绑定一个处理函数，或者触发元素上的 change 事件。

这个函数的第一种用法是 .bind('change', handler) 的快捷方式，第二种用法是 .fire('change') 的快捷方式。

参数：

fn
可选。Funcion。回调函数。
例子 
            $.require("ready,event,fx",function(){
                $("#sweets").change(function () {
                    var str = "";
                    $("option:selected",this).each(function () {
                        str += $(this).text() + " ";
                    });
                    $("#show_result").text(str);
                })
                .change();
                
                $('body').delegate('.listOfOptions', 'change', function() {
                    var selected = $(this).find(':selected')
                    if (selected.prop('class') == 'customOption') {
                        $("#show_result2").text(selected.text()+" ！！！");
                    } else {
                        $("#show_result2").text(selected.text());
                    }
                });
                $("#checkbox").change(function(){
                    $("#show_result3").text(this.checked)
                });
            });;
