$.fn.undelegate(selector, types, fn, [times])

描述：

移除通过delegate绑定的事件回调。

参数：

如果只有一个字符串参数，则selector意为types，如果不小心写错为选择器， 它会抛出“hash.type should be a combination of this event type and the namespace”异常

如果只有一个对象参数，则其会分解为type与回调。

如果是一个对象参数跟一个字符串参数，则那个字符串会被认为是选择器。

否则情况同函数定义。

例子
Click!
$.require("ready,event,fx", function() {
    function callback() {
        $(".hide_div").show(1000);
    }
    $("#bind").click(function() {
        $("body").delegate("#theone", "click", callback)
        $("#theone").html("现在点击它下方会出现文字");
    });
    $("#unbind").click(function() {
        $("body").undelegate("#theone", "click")
        $("#theone").html("现在点击它一点反应也没有");
        $(".hide_div").hide(1000);
    });
});
