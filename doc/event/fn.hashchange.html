$.fn.hashchange([ fn ])

描述：

当hash发生改变时触发此事件，解决AJAX下后退按钮失灵的问题。

此事件是单独在一个模块中，在more/hashchange.js中。这样做完全是因为它的存在性并不重要，只是一个补丁模块。当IE678淡出舞台，它就没有存在的必要。

需要注意的是,此事件只能绑在window上。

参数：

fn
可选。Funcion。回调函数。
例子 
$.require("ready,more/hashchange", function() {
    var el = $("#show_result")
    document.onclick = function() {
        var t = "#" + Math.random();
        location.hash = t
    }
    $(window).hashchange(function() {
        el.html(location.href)
    })

})
