$.fn.sroll([ fn ])

描述：

为 sroll 事件绑定一个处理函数，或者触发元素上的 sroll 事件。

这个函数的第一种用法是 .bind('sroll', handler) 的快捷方式，第二种用法是 .fire('sroll') 的快捷方式。

此事件适应于window以及所有带滚动条的元素节点

参数：

fn
可选。Funcion。回调函数。
例子 
$.require("ready,event,fx", function() {
    $("p").clone().appendTo("body");
    $("p").clone().appendTo("body");
    $("p").clone().appendTo("body");
    $(window).scroll(function() {
        $("span").show().fadeOut(2000);
    });
})
