$.fn.bind("inview", fn)

描述：

这是一个非常有用的自定义事件,用于判定某元素位于可视区之内(外)。

注,此方法是独立一个新模块,位于more目录下。

$(expr).bind('inview', function(event, visible, status) {
    if (visible == true) {
        $.log("当前元素进入可视区,管他显示完整不完整")
        if (status == 'top') {
            $.log("当前元素的上部可见")
        } else if (status == 'bottom') {
            $.log("当前元素的下部可见")
        } else {
            $.log("当前元素整体可见")
        }
    } else {
        $.log("当前元素离开可视区")
    }
});

回调函数的参数：

event
事件对象
visible
布尔值,true或者false.
status
字符串,表示当前的可视情况,分别为top, bottom, both
例子 
$.require("ready,event,more/inview", function() {
    var logger = $("#logger")
    $("#sample").bind('inview', function(event, visible, status) {
        if (visible == true) {
            if (status == 'top') {
                logger.text("当前元素的上部可见")
            } else if (status == 'bottom') {
                logger.text("当前元素的下部可见")
            } else {
                logger.text("当前元素整体可见")
            }
        } else {
            logger.text("当前元素离开可视区")
        }
    });
});
