$.fn.focusout( fn )

描述：

为 focusout 事件绑定一个处理函数。

这个函数是 .bind('focusout', handler) 的快捷方式。

focusout 事件会在元素（或者其内部的任何元素）失去焦点时触发。这跟 blur 事件的显著区别在于， 它可以在父元素上检测子元素失去焦点的情况(换而言之，它支持事件冒泡)。

参数：

fn
必需。Funcion。回调函数。
例子 
$.require("ready,event,fx", function() {
    var fo = 0,
        b = 0;
    $(".inputs p").focusout(function() {
        fo++;
        $("#fo").text("focusout fired: " + fo + "x");
    }).blur(function() {
        b++;
        $("#b").text("blur fired: " + b + "x");
    });
});
