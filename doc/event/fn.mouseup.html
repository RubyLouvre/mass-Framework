$.fn.mouseup([ fn ])

描述：

为 mouseup 事件绑定一个处理函数，或者触发元素上的 mouseup 事件。

这个函数的第一种用法是 .bind('mouseup', handler) 的快捷方式，第二种用法是 .fire('mouseup') 的快捷方式。

当鼠标指针在元素上和按下鼠标键，mouseup事件被发送到这个元素。任何HTML元素都可以收到此事件。

任何鼠标按钮被点击时mouseup事件就会被发送。作用于特定的按钮，我们可以使用事件对象的which属性。并非所有浏览器都支持这个属性（IE浏览器使用button属性代替）， dom Framework对于做了统一处理，对于任何浏览器。点击鼠标左键which的值是1，点击鼠标中键which的值是2，点击鼠标右键which的值是3。

参数：

fn
可选。Funcion。回调函数。
例子

测试onmousedown与onmouseup的执行顺序.

$.require("ready,event", function() {
    $(".sample p").mouseup(function() {
        $(this).append('<span style="color:#F00;">Mouse up.</span>');
    }).mousedown(function() {
        $(this).append('<span style="color:#00F;">Mouse down.</span>');
    });
});

Press mouse and release here.
