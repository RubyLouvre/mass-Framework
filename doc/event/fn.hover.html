$.fn.hover(fnIn, [fnOut])

描述：

将二个事件函数绑定到匹配元素上，分别当鼠标指针进入和离开元素时被执行。如果只有一个函数，它会兼职第二个参数的工作。

.hover()方法是同时绑定 mouseenter和mouseleave事件。我们可以用它来简单地应用在鼠标在元素上活动行为。

调用$(selector).hover(fnIn, fnOut)是以下写法的简写：

$(selector).mouseenter(fnIn).mouseleave(fnOut || fnIn);

更多细节参见 .mouseenter()和 .mouseleave()。

参数：

type
必需。String.事件类型。
args0
额外参数
args1
额外参数
argsn
额外参数
例子 
$.require("ready,event,fx", function() {
    $(".sample p").click(function() {
        $(this).slideUp(function() {
            if (!$(".sample p:visible").length) {
                $(".sample p").delay(1000).show(1000)
            }
        });
    });
    $(".sample p").hover(function() {
        $(this).toggleClass("hilite");
    });
});
