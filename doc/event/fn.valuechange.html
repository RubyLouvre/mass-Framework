$.fn.valuechange([ fn ])

描述：

行为有点类似change事件，不过change是在失去焦点才拿输入前的值与现在的值作比较，而valuechange则在整个输入过程都在比较。它有点类似于input事件， 但它比input更厉害之外在于，它能在监控使用el.setAttribute("value","zz"),el.value = "xxx"及使用语音输入手段引发的值改变。

回调函数的第一值参数为事件对象，比一般的事件对象多出三个属性：

  • oldValue: 旧的值。
  • newValue: 当前值。
  • realType: 是由哪种原生事件触发的。

需要注意的是,它并不在event.js文件,而在more/valuechange.js中。

参数：

fn
可选。Funcion。回调函数。
例子 
$.require("ready,more/valuechange", function() {
    $("#aaa").valuechange(function(e) {
        var str = e.oldValue + " " + e.newValue + " " + e.realType
        $("#show_result").text(str);
    })
});;
