API 解说
css 取得第一个元素节点的样式或为所有选中的元素节点设置样式
offset 取得第一个元素节点位于页面的坐标或为所有选中的元素节点设置坐标
position 取得第一个元素节点相对于其offsetParent的坐标
offsetParent 在当前找到的节点中，收集它们的offsetParent，封装成mass对象返回
scrollParent 在当前找到的节点中，收集它们的scrollParent，封装成mass对象返回
scrollTop 取得第一个元素节点的scrollTop或为所有选中的元素节点设置scrollTop
scrollLeft 取得第一个元素节点的scrollLeft或为所有选中的元素节点设置scrollLeft
width 取得第一个元素节点的width(即使它被隐藏了)或为所有选中的元素节点设置width
height 取得第一个元素节点的height(即使它被隐藏了)或为所有选中的元素节点设置height
innerWidth 取得第一个元素节点的width与它左右padding之和
innerHeight 取得第一个元素节点的height与它上下padding之和
outerWidth 取得第一个元素节点的width与它左右padding,左右border之和（如果传参为true，则还包括其margin）
outerHeight 取得第一个元素节点的height与它上下padding,上下border之和（如果传参为true，则还包括其margin）
show 让元素显示出来
hide 让元素隐藏起来 
$("#aaa").css("background-color", "red")

$("#aaa").css("bgc", "blue")
