$.fn.offsetParent()

描述：

返回所有匹配节点的offsetParent所组成的新dom对象。

例子

寻找类名为tem-a的节点的offsetParent：


      <ul class="level-1">
      <li class="item-i">I</li>
      <li class="item-ii" style="position: relative; background-color: red;">II
        <ul class="level-2">
          <li class="item-a">A</li>
          <li class="item-b">B
            <ul class="level-3">
              <li class="item-1">1</li>
              <li class="item-2">2</li>
              <li class="item-3">3</li>
            </ul>
          </li>
          <li class="item-c">C</li>
        </ul>
      </li>
      <li class="item-iii">III</li>
    </ul>
                
            $.require("ready,node,css,event",function(){
                $("li.item-a").offsetParent().css("background-color":"#FF6347");
            });
