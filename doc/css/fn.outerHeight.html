$.fn.outerHeight( [includeMargin] )

描述：

取得第一个元素的当前高度,包括padding，border和选择性的margin

此方法是个只读文法。

此方法不适用于window 与 document对象，可以使用.height()代替。

参数：

includeMargin
可选。Boolean。是否加上margin。
例子 

    
    
 
 
$.require("ready,node,css", function() {
    var a = $(".outerHeight");
    a.html("它的outerHeight为" + a.outerHeight() + "px");
    a.html("<br/>它的outerHeight(包括margin)为" + a.outerHeight(true) + "px");
});
