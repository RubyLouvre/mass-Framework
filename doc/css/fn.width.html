$.fn.width( [val])

描述：

取得或设置匹配节点的宽。

此方法是属性于get one set all操作，根据传入参数决定其是读操作还是写操作。

如果没有参数，则读取第一个元素的width，并返回。

如果存在参数，通过它设置所有元素的width,并返回原实例。

.css(width) 和 .width()之间的区别是后者返回一个没有单位的数值（例如，400），前者是返回带有完整单位的字符串（例如，400px）。当一个元素的宽度需要数学计算的时候推荐使用.width() 方法 。

如果没有设置单位，默认为px。

它可以获取隐藏元素的宽。

这是测试的小白鼠

$.require("ready,node,css,event", function() {
    function showHeight(ele, h) {
        $("#show_result").text(ele + h + "px.");
    }
    $("#getp").click(function() {
        showHeight("段落的宽为", $("#test_p").width());
    });
    $("#getd").click(function() {
        showHeight("页面的宽为", $(document).width());
    });
    $("#getw").click(function() {
        showHeight("窗口的宽为", $(window).width());
    });
})
