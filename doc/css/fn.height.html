$.fn.height( [val])

描述：

取得或设置匹配节点的高。

此方法是属性于get one set all操作，根据传入参数决定其是读操作还是写操作。

如果没有参数，则读取第一个元素的height，并返回。

如果存在参数，通过它设置所有元素的height,并返回原实例。

.css('height') 和 .height()之间的区别是后者返回一个没有单位的数值（例如，400），前者是返回带有完整单位的字符串（例如，400px）。当一个元素的高度需要数学计算的时候推荐使用.height() 方法 。

如果没有设置单位，默认为px。

它可以获取隐藏元素的高。

w3c的盒子模型

例子
 

这是测试的小白鼠

$.require("ready,node,css,event", function() {
    function showHeight(ele, h) {
        $("#show_result").text(ele + h + "px.");
    }
    $("#getp").click(function() {
        showHeight("段落的高为", $("#test_p").height());
    });
    $("#getd").click(function() {
        showHeight("页面的高为", $(document).height());
    });
    $("#getw").click(function() {
        showHeight("窗口的高为", $(window).height());
    });
})
