object.clone()

描述：

对对象进行深拷贝。

例子 
                $.require("lang",function(){
                    var obj = {
                        e:1,
                        b:"aaa",
                        c:[1,2,3],
                        d:{
                            bb:"bb"
                        }
                    };
                    $.log($.lang(obj).clone(),true);
                });
