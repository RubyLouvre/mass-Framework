array.clone()

描述：

对数组进行深拷贝。

例子 
                $.require("lang",function(){
                    var arr = [3,5,6,1,[20,2,[1,0],7]]
                    var ret = $.lang(arr).clone()
                    ret = $.dump(ret);//序列化
                    $.log(arr === ret,true);
                    $.log(ret,true);
                })
