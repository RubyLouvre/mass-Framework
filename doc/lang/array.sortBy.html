array.sortBy(fn,scope)

描述：

将数给元素与其索引值传一个迭代器中，用得到的新数组进行排序。

参数：

fn
可选。回调函数，sortBy方法会为它传入两个参数，元素自身与其索引值
score
可选。要绑定的作用域。
例子 
                $.require("lang",function(){
                    var test = $.lang(["zero", "one", "two", "three"]);
                    var ret = test.sortBy(function(value){
                        return value.length;
                    });
                    $.log($.dump(ret),true);
                });
            
                $.require("lang",function(){
                    var test = $.lang([[0, 10], [1, 9], [2, 8], [3, 7]]);
                    var ret = test.sortBy(function(value, index){
                        return value[1];
                    });
                    $.log($.dump(ret),true);
                });
