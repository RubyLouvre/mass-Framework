标签伪类，不推荐使用，只是为了兼容jQuery。

  • :header匹配h1,h2,h3,h4,h5,h6。
  • :button匹配button与<input type="button"/>。
  • :link匹配链接(a标签)
  • :text匹配<input type="text"/>
  • :radio匹配<input type="radio"/>
  • :checkbox匹配<input type="checkbox"/>
  • :file匹配<input type="file"/>
  • :password匹配<input type="password"/>
  • :submit匹配<input type="submit"/>
  • :image匹配<input type="image"/>
  • :reset匹配<input type="reset"/>
  • :input匹配所有input标签
例子
请点击下面表格
$(":text").css("bgc","blue");
$(":radio").css("border","1px solid green");
$(":checkbox").css("border","1px solid red");
$(":button").css("bgc","#FF6363");
