内容包含伪类，:contains(expr)。如果元素含有expr所代表的文本内容，则会被选中，注意其最外层的引号是没有用，它大小写敏感。

例子
请点击下面表格
$(".sample div:contains('John')").css("text-decoration", "underline"); 
< div class = "sample" > < div > John Resig < /div>

 < div > Malcom John Sinclair < /div>
        
J. Ohn
