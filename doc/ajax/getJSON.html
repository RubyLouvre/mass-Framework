$.getJSON(url, data?, callback)

描述：

用于发出JSONP方法。

参数：

url
必需。请求地址。
data
可选。一个参数对象
callback
回调
例子 
          require("ajax", function() {
                $.getJSON("/javascript",{
                        aaa: 111111,
                        ccc: 22222
                    }, function(json) {
                        $.log(json);
                    })
            })

相当于

          require("ajax", function() {
                $.ajax({
                    type: "GET",
                    url: "/javascript",
                    data: {
                        aaa: 111111,
                        ccc: 22222
                    },
                    success: function(json) {
                        $.log(json);
                    },
                    dataType: "jsonp"
                })
            })
